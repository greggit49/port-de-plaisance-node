Dashboard

Fonctionnalités

Sur la page du dashboard il y a une barre pour pour voir les différente liste de catways, de réservation et voir les utilisateuret aussi il y a ensuite 3 formulaire pour gérer les catways, les réservations et les utilisateurs.
Le premier formulaire permet de gérer les Utilisateurs, il permet de créer, modifier, supprimer et consulter les Utilisateurs.
Le deuxième formulaire permet de gérer les catways, il permet de créer, modifier, supprimer et consulter les catways.
Et le troisième formulaire permet de gérer les réservations, il permet de créer, modifier, supprimer et consulter les réservations.
Il suffit de remplir les champs du formulaire et de cliquer sur le bouton correspondant à l'action que vous voulez effectuer, il va ensuite compléter les diférente liste de catways, de réservation et d'utilisateurs.
